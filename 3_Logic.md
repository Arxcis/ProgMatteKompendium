# 3. Logic

## 3.1 Bitwise operations

| Operation  | Symbol      | Signature     |
| ---------- | ----------- | ------------- |
| And        | &           | 0 - 0 - 0 - 1 |
| Or         | \|          | 0 - 1 - 1 - 1 |
| Xor        | ^           | 0 - 1 - 1 - 0 |
| Not        | ~  (not or) | 1 - 0 - 0 - 0 |
| ShiftLeft  | <<          | <<1 == 10     |
| ShiftRigth | \>>         | \>>1 ==  0    |
|            |             |               |
